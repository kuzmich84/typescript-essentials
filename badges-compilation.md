## Reflections

1. **Getting Started with TypeScript**: I learned in this module, why TypeScript was created and how to use it fo JS development.
2. **Declare Variable Types in TypeScript**: This module introduces some of the available types and shows how to associate them with variables. I learned how the static type system in TypeScript allowing it to validate that my code is working correctly.
3. **Implement Interfaces in TypeScript**: In this module I learned how to implement interfaces in TypeScript and how I can use interfaces to define object types.
4. **Develop Typed Functions in TypeScript**: TypeScript adds some new capabilities to the standard JavaScript functions to make them easier to work with. TS allows you to determine the type of input parameters for a function and determine the type of value returned by the function.
5. **Declare and Instantiate Classes in TypeScript**: I learned which featuгes TS adding to classes. Classes in TypeScript extend the ES6 functionality by adding TypeScript-specific features like type annotations for class members, access modifiers (public, protected, private), and the ability to specify required or optional parameters.
6. **Generics in TypeScript**: Generics are code templates that I can define and reuse throughout in my code. They provide a way to tell functions, classes, or interfaces what type you want to use when you call it.
7. **Work with External Libraries in TypeScript**: I learned how use modules in TS. It help me control which code is exposed to the global namespace of my applications.
8. **Organize Code with Namespaces in TypeScript**: I learned how use namespaces in TS. Namespaces provide to organize code how modules - the code is exposed to the global namespace. Nowaday I'm using modules instead of namespaces.
